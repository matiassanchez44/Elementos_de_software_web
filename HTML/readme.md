                       Cuestionario

1. Consulte y explique que significa a nivel de uso del internet:
a. Servidor DNS
b. IP
c. Protocolo
d. Nombre de dominio
e. Directorio y nombre de archivo

2. Mencionamos dos roles importantes en el desarrollo de un producto web. pero hay otros roles involucrados, consulte y defina que hace cada uno de los siguientes roles:

a. Diseñador UI/UX
b. Cloud Manager/ Ingeniero de infraestructura en la nube
c. Administrador de base de datos 
d. Project Manager/ Director del proyecto

3. ¿Que significa que un lenguaje sea turing completo?

4. ¿Por que CSS y HTML no son lenguajes de programacion?

5. Defina con sus palabras que es una etiqueta de HTML 

                      Solucion

1- a. Servidor DNS
Un servidor DNS es el que se encarga de traducir los nombres de las páginas web a direcciones IP. es decir, cuando uno escribe una página como google.com, el DNS busca cuál es su dirección numérica para poder acceder a ella

b. IP
La IP es un número único que identifica a cada dispositivo en Internet. Es como una dirección que permite que los equipos se encuentren entre sí en la red

c. Protocolo
Un protocolo es un conjunto de reglas que permiten que los dispositivos se comuniquen entre sí, por ejemplo, gracias a estos protocolos es posible que podamos navegar por páginas web

d. Nombre de dominio
El nombre de dominio es el nombre que usamos para entrar a una página web, como facebook.com. Es más fácil de recordar que una dirección IP

e. Directorio y nombre de archivo
El directorio es como una carpeta dentro de una página web donde se guardan archivos, y el nombre de archivo es el nombre específico de uno de esos archivos. Juntos forman la ruta completa de un recurso en internet

2-  Roles en el desarrollo de un producto web
a. Diseñador UI/UX

El diseñador UI/UX se encarga de la apariencia y la experiencia del usuario. Yo entiendo que UI se enfoca en cómo se ve la aplicación: colores, botones, mientras que UX se enfoca en que sea fácil de usar y agradable

b. Cloud Manager / Ingeniero de infraestructura en la nube

Se encarga de toda la parte técnica en la nube, como los servidores y servicios donde funciona la aplicación. También asegura que todo funcione bien, sea seguro y soporte muchos usuarios

c. Administrador de base de datos

Es quien gestiona la base de datos, organiza la información y se asegura de que esté segura y disponible cuando se necesite

d. Project Manager / Director del proyecto

Es la persona que organiza todo el proyecto, coordina al equipo y se asegura de que se cumplan los tiempos y objetivos

3. ¿Qué significa que un lenguaje sea Turing completo?

Significa que un lenguaje puede resolver cualquier problema computacional si se le dan los recursos necesarios, es decir, puede ejecutar cualquier tipo de lógica o cálculo

4. ¿Por qué CSS y HTML no son lenguajes de programación?

Porque no tienen lógica como tal HTML sirve para estructurar el contenido y CSS para darle estilo, pero no pueden tomar decisiones ni ejecutar instrucciones como lo haría un lenguaje de programación

5. ¿Qué es una etiqueta de HTML?

Una etiqueta de HTML es una forma de marcar el contenido para indicarle al navegador cómo debe mostrarse por ejemplo, una etiqueta puede indicar que un texto es un párrafo, un título o una imagen
